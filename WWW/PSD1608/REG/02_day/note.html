正则表达式的应用 匹配,查找,分割,替换 对于普通字符串处理函数和正则表达式字符串处理函数,使用的原则: 能对于普通字符串处理函数处理的,就不要用正则表达式处理 原因: 普通字符串处理函数:效率高.功能简单 正则表达式处理函数:效率相对低.复杂.功能强大 匹配,查找 strstr(string $haystack,string $needle) 功能: 查找needle在haystack中出现后的字符串 strpos(string $haystack,string $needle) 功能: 在haystack查找needle首次出现的位置 strrpos(string $haystack,string $needle) 功能: 在haystack查找needle最后一次出现的位置 substr(string $str,int $start [,int $length]) 功能:字符串的截取 str 原字符串 start 开始截取的位置 length 截取长度,默认截取从start起到最后 preg_match_all(string $pattern,string $subject,array $match); 功能:正则匹配 返回:匹配到的所有结果 preg_match(string $pattern,string $subject,array $match) 功能:正则匹配 返回:仅一次匹配到的结果(0~1) preg_grep(string $pattern,array $input) 功能:正则匹配 pattern 正则 input 要进行匹配的字符串数组. 返回: 匹配到的结果,返回格式:数组 分割 explode(string $delimiter,string $str[,int $limit]) 功能:分割 参数: delimiter 分割的标识字符串 str 要进行分割的字符串 limit 分割的次数 返回:分割好的字符串.格式:数组 注意:delimiter不能为空 implode:为连接 preg_split(string $pattern,string $subject) 功能:字符串的分割 pattern 正则 subject 要进行分割的字符串 返回:被分割好的字符串 替换 str_replace(mixed $search,mixed $replace,mixed $str) 功能:在str字符串中替换 preg_replace(string $pattern,string $replace,string $subject) 功能:字符串的替换 贪婪模式与非贪婪模式 贪婪模式 .* 非贪婪模式 .*?
