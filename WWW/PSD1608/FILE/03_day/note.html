file_put_contents(string $filename,mixed $date) 功能:向文件中写入内容 参数: filename 文件地址 data 写入的内容 返回: 成功:写入的字节数. 失败:flase 注意: 1.破坏原有文件的内容 2.若写入文件不存在,尝试创建 fopen(string $filename,string $mode) 功能:打开文件 参数: filename 要打开文件地址 mode 以何种方式打开文件 read r 以只读方式打开文件,若文件不存在,不尝试创建. r+ 以读写方式打开文件,若文件不存在,不尝试创建. write w 以只写方式打开文件,将原有文件内容清空,写入新内容,若文件不存在,尝试创建. w+ 以读写方式打开文件,将原有文件内容清空,写入新内容,若文件不存在,尝试创建. append a 以只写方式打开文件,在原有文件内容后追加写入新内容,若文件不存在,尝试创建. a+ 以读写方式打开文件,读取是从文件首读取,追加是内容后追加写入新内容,若文件不存在,尝试创建. 返回: 文件资源 fwrite(resource $fh,string $data) 功能:在已经打开的文件中写入内容 参数: fh 申请的文件资源 data 写入的内容 返回:成功:写入的字节数.成功写入的字节数有可能是0 失败:flase fget(resource $fh) 功能: 获取已经打开文件中的一个字符 参数: fh 申请的文件资源 返回: 读取的文件字符.一次仅获取一个字符.获取不到返回flase fclose(resource $fh) 功能:释放申请的文件资源 参数: fh 申请的文件资源 返回:成功 true 失败 flase fgets(resource $fh) 功能:从已经打开的文件中读取一行 参数: fh 申请的文件资源 返回:读取的文件字符.一次仅获取一行. 获取不到返回flase fread(resource $fh,int $length) 功能:从已经打开的文件中读取指定长度内容 feof(resource $fh) 功能:判断文件指针是否到达文件尾
